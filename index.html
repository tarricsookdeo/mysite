Django Auth API

Django is my favorite framework for building web apps and robust API's. Django comes with a lot of features built it, and has an expansive selection of 3rd party apps and libraries to get the job done. One of the most common tasks in any app is handling user authentication. The dj-rest-auth package for Django helps create an authentication API. However, there's quite a lot of configuration and tweaking needed to get this setup to play nice with a frontend framework such as React or Vue. This project is meant to be a demo for configuring Django, along with dj-rest-auth to be useable with an SPA.

Built using

  • Django
  • Django Rest Framework
  • Django Rest Auth
